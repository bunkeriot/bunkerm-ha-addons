# BunkerM — Home Assistant Add-on

[![HA Add-on](https://img.shields.io/badge/Home%20Assistant-Add--on-blue)](https://www.home-assistant.io/addons)

Full MQTT broker management platform for Home Assistant, powered by Eclipse Mosquitto.

## Features

- 🦟 **Eclipse Mosquitto** MQTT broker on port 1883
- 🖥️ **Web dashboard** on port 2000 — clients, ACLs, roles, groups
- 📊 **Real-time monitoring** — broker stats, $SYS topics, connection events
- 🤖 **AI assistant** — natural language queries about your broker (BunkerAI Cloud)
- 🔐 **Dynamic security** — full ACL management without restarting the broker

## Installation

Add this repository to Home Assistant:

```
https://github.com/BunkerM/bunkerm-ha-addons
```

See [DOCS.md](bunkerm/DOCS.md) for full setup instructions.
